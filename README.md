# 잡데렐라 (Cinderella for Jobs)

"신데렐라처럼 멋진 직업(Job)을 찾아주는 동화 같은 앱" — Streamlit 기반 데모 앱입니다.

주요 기능
- 이력서(PDF / Word) 업로드 및 텍스트 추출
- 이력서 기반 키워드 추출 및 추천 채용 공고 제안(모의 데이터)
- 이력서 개선 피드백, 직무 인터뷰/합격자소서 예시, 콘텐츠LAB

설치 및 실행
1. 의존성 설치

```bash
pip install -r requirements.txt
```

2. 앱 실행

```bash
streamlit run streamlit_app.py
```

설명: 이 저장소는 데모 목적의 기본 구현을 포함합니다. 실제 잡코리아 연동이나 이메일 전송 등은 현재 모의(샘플)로 동작합니다.
